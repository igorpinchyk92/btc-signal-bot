# Grok Crypto Swing Bot — Уровни BTC каждые 30 минут

**Автоматический бот от Grok** — присылает в твою Telegram-группу актуальные уровни Bitcoin каждые 30 минут 24/7.

### Что делает бот:
- Цена BTC в реальном времени  
- Ключевые уровни: поддержка, сопротивление, POC, FVG  
- Объём 24ч и изменение  
- Мой (Grok) вывод: LONG bias / SHORT bias  
- Работает **без твоего участия** — на серверах GitHub (бесплатно, навсегда)
